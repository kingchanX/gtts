# Google Text-to-Speech Application

모던 UI를 갖춘 Python 기반 텍스트-음성 변환(TTS) 애플리케이션입니다.

## 주요 기능

- 다양한 언어 지원 (영어, 한국어, 일본어, 중국어 등)
- 변환 속도 조절 가능
- 미리 듣기 기능
- 다중 텍스트 항목 지원
- MP3 형식으로 저장

## 필요 라이브러리

- gTTS (Google Text-to-Speech)
- tkinter
- pygame

## 설치 방법

```bash
pip install gtts pygame
```

## 사용 방법

```bash
python gtts_app.py
```
